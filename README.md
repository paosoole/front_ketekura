
# Ketekura - React Frontend


Este repositorio contiene un frontend en React (Vite) inspirado en el diseño proporcionado en Canva: https://vistausariofullstack.my.canva.site/ketekura


Funcionalidad principal:
- Consultar pacientes, médicos, atenciones e información de pagos
- Conecta a un backend (microservicio Spring Boot) que expone APIs REST para las entidades relacionales (Oracle) y no relacionales (MongoDB)


Stack recomendado:
- React + Vite
- Tailwind CSS
- react-router-dom
- Fetch API (puedes reemplazar con Axios)
